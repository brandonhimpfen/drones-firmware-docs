# Bench Testing

Bench tests validate firmware behavior on hardware before flight.

## Bench test areas

- Boot and firmware version checks
- Sensor detection
- Parameter load/save
- Calibration persistence
- Telemetry link
- Logging start/stop
- Motor output inhibition while disarmed
- Actuator direction checks
- Failsafe trigger simulation
- Firmware update and rollback

## Bench safety

Use a safe work area, stable power, and clear physical controls. Propellers should be removed for routine bench testing.
