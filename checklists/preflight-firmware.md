# Firmware Pre-Flight Checklist

- [ ] Firmware version matches approved version
- [ ] Hardware target matches aircraft
- [ ] Parameter profile matches operation
- [ ] Parameter hash recorded
- [ ] Sensors healthy
- [ ] Calibration current
- [ ] Battery thresholds configured
- [ ] Failsafe actions configured
- [ ] Geofence loaded, if required
- [ ] Mission uploaded and verified, if required
- [ ] Logging enabled
- [ ] Telemetry link verified
- [ ] Arming checks pass
