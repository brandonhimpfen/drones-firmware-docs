# Geofencing Documentation

Geofencing constrains where a drone may operate.

## Geofence fields

- Fence type
- Coordinate frame
- Horizontal boundary
- Altitude boundary
- Breach detection rate
- Warning threshold
- Breach action
- Override policy
- Logging fields

## Required tests

- Valid fence load
- Invalid fence rejection
- Boundary warning
- Boundary breach action
- Fence persistence after reboot
- Behavior during GPS degradation
