# Actuator and Motor Output Documentation

Actuator documentation defines how control demands become physical outputs.

## Documented elements

- Motor ordering
- Servo channels
- Output protocol
- Update rate
- Min/max command values
- Failsafe output values
- Arming/disarming behavior
- Mixer or allocation matrix
- ESC calibration expectations

## Safety expectations

Motor tests and actuator checks must be designed for controlled bench environments. Remove propellers before bench motor testing unless the test procedure explicitly requires otherwise and is performed by qualified personnel in a controlled setting.
