# HIL and SITL Testing

Simulation-in-the-loop and hardware-in-the-loop testing help validate behavior before flight.

## SITL use cases

- Mission logic validation
- Mode transition testing
- Telemetry compatibility
- Parameter regression testing
- Failsafe scenario rehearsal

## HIL use cases

- Board-specific timing validation
- Sensor interface validation
- Actuator output verification
- Firmware update validation
- Watchdog and reset behavior

## Evidence to retain

- Firmware version
- Simulator version
- Vehicle model
- Parameter set
- Test scenario
- Logs
- Pass/fail result
- Known deviations
